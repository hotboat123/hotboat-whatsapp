# 📸 Guía de Imágenes para WhatsApp

## 📁 Estructura de Carpetas

```
media/images/
├── alojamientos/       → Imágenes de alojamientos (Open Sky, Raíces de Relikura)
├── experiencias/       → Imágenes de experiencias (Rafting, Cabalgatas, Navegación)
└── packs/
    ├── pack_1_noche/   → Imágenes para paquete de 1 noche
    ├── pack_2_noches/  → Imágenes para paquete de 2 noches
    └── pack_3_noches/  → Imágenes para paquete de 3 noches
```

## 📝 Instrucciones

### 1️⃣ Formato de Imágenes
- **Formato recomendado:** `.jpg` (mejor compresión, carga más rápido)
- **También soportado:** `.png`, `.jpeg`, `.webp`
- **Tamaño óptimo:** Ancho máximo 1080px (WhatsApp las redimensiona de todos modos)
- **Peso recomendado:** Menos de 500KB por imagen (para carga rápida)

### 2️⃣ Nombres de Archivos
- **No importa el nombre** - El sistema envía todas las imágenes de cada carpeta
- **Orden alfabético** - Las imágenes se envían en orden alfabético
- Ejemplo: `01-intro.jpg`, `02-detalles.jpg`, `03-precios.jpg`

### 3️⃣ Cuántas Imágenes Poner

#### Alojamientos (`alojamientos/`)
- **Recomendado:** 3-5 imágenes
- Mostrar: Open Sky Domos, Raíces de Relikura Cabañas, Hostal, precios

#### Experiencias (`experiencias/`)
- **Recomendado:** 4-6 imágenes
- Mostrar: Rafting, Cabalgatas, Navegación, precios de cada experiencia

#### Packs (`packs/pack_X_noche/`)
- **Recomendado:** 2-4 imágenes por pack
- Mostrar: Itinerario, actividades incluidas, precio total, condiciones

### 4️⃣ Consejos de Diseño
- ✅ Texto grande y legible (la gente las verá en el celular)
- ✅ Información clara y concisa
- ✅ Colores de marca (azul/celeste para HotBoat)
- ✅ Incluir emojis para hacer más visual
- ✅ Precio destacado
- ❌ No llenar con mucho texto (mejor varias imágenes simples)

## 🚀 Cuándo Se Envían

- **Experiencias:** Cuando el usuario selecciona opción 6 (Otras Experiencias Pucón)
- **Alojamientos:** Cuando el usuario selecciona opción 2 del submenú de alojamientos
- **Packs:** Cuando el usuario elige 1, 2 o 3 noches del menú de paquetes completos

## 🔄 Actualizar Imágenes

1. **Reemplaza** las imágenes en la carpeta correspondiente
2. **Reinicia** el servidor (si está corriendo localmente)
3. **Listo** - Las nuevas imágenes se enviarán automáticamente

## ⚠️ Importante

- Las imágenes se envían **en orden alfabético**
- Solo la **primera imagen** lleva caption (pie de foto)
- Si una carpeta está **vacía**, el bot enviará un mensaje de error
- El sistema envía **todas las imágenes** de la carpeta (no limites)
