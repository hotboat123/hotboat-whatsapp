## 🎨 Especificaciones Técnicas

### Formato
- **Extensión:** `.jpg` (recomendado), también `.png`, `.jpeg`, `.webp`
- **Ancho:** 1080px máximo (WhatsApp optimiza automáticamente)
- **Peso:** Menos de 500KB por imagen
- **Orientación:** Vertical preferiblemente (más natural en móviles)

### Nombres de Archivos
- Usar números para ordenar: `01-portada.jpg`, `02-incluye.jpg`, `03-precios.jpg`
- El sistema envía en orden alfabético
- El nombre específico no importa, solo el orden

### Diseño

#### Para Pack Romántico 💕
- Colores: Rosa, rojo, dorado
- Emojis: 💕 🌹 ❤️ 🥂 ✨
- Estilo: Elegante, íntimo, romántico

#### Para Pack Familiar 👨‍👩‍👧‍👦
- Colores: Verde, azul, amarillo (alegres)
- Emojis: 👨‍👩‍👧‍👦 🎈 🍦 🎯 🏡
- Estilo: Divertido, seguro, familiar

#### Para Pack Amigos 👥
- Colores: Naranja, azul, verde (energéticos)
- Emojis: 🎉 🍻 🎵 🔥 💪
- Estilo: Aventurero, festivo, dinámico

### Elementos Comunes
- ✅ Texto GRANDE y legible
- ✅ Logo HotBoat
- ✅ Emojis para hacer visual
- ✅ Información clara y concisa
- ✅ Precios destacados
- ✅ Call-to-action claro
- ❌ Evitar párrafos largos de texto

## 💡 Tips Pro

1. **Empieza con lo más popular:** Probablemente Pack Romántico y Pack Familiar sean los más solicitados
2. **Reutiliza elementos:** Las portadas pueden tener un diseño base común, solo cambiando colores y fotos
3. **Premium = Más completo:** Las versiones premium deben verse claramente "más completas" que las básicas
4. **Diferenciación clara:** Cada pack debe tener su propia identidad visual

## 🚀 ¿Cuándo se muestran?

Cuando actualicemos el código, el flujo será:
1. Usuario selecciona "Paquetes Completos"
2. Bot muestra: Romántico, Familiar, Amigos
3. Usuario elige tipo (ej: "Romántico")
4. Bot pregunta: Básico o Premium
5. Bot envía las imágenes correspondientes
