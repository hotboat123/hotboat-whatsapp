# 💬 Cómo Preguntar por Disponibilidad en WhatsApp

## ✅ Mensajes que SÍ funcionan

El bot detecta automáticamente cuando preguntas por disponibilidad. Usa cualquiera de estas frases:

### 📅 Consultas generales:
```
¿Tienen disponibilidad?
¿Hay disponibilidad?
¿Están disponibles?
disponibilidad
```

### 📆 Consultas con tiempo específico:
```
¿Tienen disponibilidad para mañana?
disponibilidad mañana
¿Hay horarios disponibles mañana?
¿Tienen disponibilidad para la próxima semana?
disponibilidad próxima semana
¿Qué días tienen disponible?
```

### 🗓️ Consultas con fechas específicas (NUEVO ✨):
```
14 de febrero
18 de noviembre
25 de diciembre
febrero 14
14/02
18-11
```

**✨ IMPORTANTE:** Ya NO necesitas decir "disponibilidad" - solo di la fecha y el bot entenderá que preguntas por disponibilidad.

Ejemplos que funcionan:
- "14 de febrero" → Te muestra disponibilidad solo para ese día
- "18 de noviembre" → Te muestra disponibilidad solo para ese día  
- "14/02" o "14-02" → Formato corto también funciona
- "¿Qué hay el 25 de diciembre?" → También funciona con preguntas

También puedes combinarlos:
- "disponibilidad 14 de febrero" → También funciona (palabra extra opcional)
- "quiero reservar el 18 de noviembre" → También funciona

### 🗓️ Consultas sobre reservas:
```
Quiero reservar
Quiero agendar
¿Puedo reservar?
¿Cuándo puedo reservar?
Necesito reservar un tour
```

### ⏰ Consultas sobre horarios:
```
¿Qué horarios tienen disponibles?
¿Hay horarios libres?
¿Cuándo tienen disponible?
¿Qué días tienen disponible?
```

## ❌ Mensajes que NO activan disponibilidad

Estos mensajes serán procesados por el FAQ o la IA:

```
Hola
¿Cuánto cuesta?
¿Dónde están ubicados?
Cuéntame sobre los tours
Información
```

## 🔑 Palabras Clave que Activan Disponibilidad

El bot busca estas palabras en tu mensaje:

- ✅ `disponibilidad`
- ✅ `disponible`
- ✅ `horario`
- ✅ `cuándo` / `cuando`
- ✅ `fecha`
- ✅ `día`
- ✅ `reservar`
- ✅ `reserva`
- ✅ `agendar`

**No importa cómo las uses**, si tu mensaje contiene alguna de estas palabras, el bot entenderá que preguntas por disponibilidad.

## 💡 Tips para mejores resultados

1. **Sé específico con el tiempo**:
   - "disponibilidad mañana" → Te muestra solo mañana
   - "disponibilidad próxima semana" → Te muestra 7 días
   - "disponibilidad mes" → Te muestra 30 días

2. **Puedes combinar palabras**:
   - "¿Qué horarios tienen disponibles para mañana?"
   - "Quiero saber la disponibilidad para reservar"

3. **El bot es inteligente**:
   - Incluso si no usas las palabras exactas, el bot puede entender la intención
   - Si dices "Quiero agendar", detectará que es sobre disponibilidad
